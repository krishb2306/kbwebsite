# Krishanth Babu Portfolio

Static portfolio site for GitHub Pages.

## Deploy

1. Push these files to the GitHub Pages repository.
2. In GitHub, open **Settings > Pages**.
3. Set the source to the main branch root.
4. Keep `CNAME` in the repository for `krishanthbabu.com`.

The resume button points to `assets/resume/Krishanth-Babu-Resume.pdf`, so replacing that file is enough to update the downloadable resume.
